Jim Quail's Gundam stuff

Welcome to my repository of Gundam stuff. On this page you can find various resources to help you access niche Gundam content. So far, this page only has translated worldbuilding entries from Advance of Zeta Re-Boot. More will be added in the future, as time and interest permits.

Additional Notes

These are the most approxmiate fonts I used based off font identification on Advance of Zeta and Re-Boot, if you want to do something similar or want to retain aesthetic consistency;
  • ITC Conduit Bold for headings and titles.
  • ITC Avante Garde Gothic Std for subtitles and "lesser" headings.
  • Hiragino Kaku Gothic Pro W3
If you need it to be even more accurate, consider cracking open the page source; the fonts they used are defined in the CSS files.
